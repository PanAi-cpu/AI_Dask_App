# About

------

**AI_Dask_app** *(暂且先叫这个名字吧，毕竟还在开发阶段)* 是一款对于**织布检验**的项目文件，是基于YOLO深度学习模型的AI验布机桌面应用，需要与硬件一起结合。能够自动检测[布匹缺陷]()（如破洞、污渍、断纱……），并记录缺陷位置对应的米数，提供实时视频监控和统计数据。

# Features

------



- 操作简易，可视化部分详细，能将织布状态实时的反应在桌面应用上
- 反应迅速，能做到实时监测
- ……

# hardware requirements

------

- 验布机支架
- 计米器
- 摄像头
- 显存大于12G

# 布匹缺陷总类



------



holes：破洞 / 孔洞 / 布面被穿破形成洞

water_oil_stains stains：水渍 / 油渍污点 / 油或水造成的污染

three wires：三丝 / 织物中混入的三根异纤维

knots：结头 / 打结 / 纱线打结形成的凸点

flower board jumps：花板跳花 / 提花机花板错误造成图案跳位

hundred feet：百脚 / 类似很多细小脚印状缺陷

hair grains：毛粒 / 毛丝 / 布面有细毛或纤维颗粒

coarse classics：粗节 / 纱线粗细不均形成粗点

Pine warp：松经 / 经纱张力不足造成松散

broken warp：断经 / 经纱断裂

hanging warp：挂经 / 经纱悬挂在布面

coarse dimension：粗纬 / 粗节纬 / 纬纱粗细异常

weft shrinkage：纬缩 / 纬纱收缩导致布面变形

pulp spot：浆斑 / 上浆残留形成斑点

warping knot：整经结头 / 整经时产生的结

star_flower jump：星花跳 / 花纹错位

broken spandex：氨纶断丝 / 弹性纱断裂

Dense_Waves_Color file：密波色档 / 密度或颜色波纹

Grinding_Crushing_Repairing_Singeing mark：磨压修烧痕 / 加工过程留下的痕迹

mixture wrinkle：混纺皱 / 布面出现褶皱
